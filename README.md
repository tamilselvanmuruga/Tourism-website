# 🌍 Tourism Website

A responsive and interactive tourism website developed to showcase popular destinations, galleries, and feedback features with a clean and user-friendly interface.

---

## 📌 Project Overview

The Tourism Website is a frontend web application designed to provide information about tourist destinations. The project focuses on responsive design, smooth navigation, and structured content presentation.

This website allows users to:

* Explore multiple tourist locations
* View image galleries
* Navigate between destination pages
* Submit feedback
* Experience responsive layout across devices

---

## 🛠️ Technologies Used

* CSS3

* Responsive Web Design
* Git & GitHub (Version Control)
* GitHub Pages (Deployment)

## 🔧 How to Run the Project Locally

1. Download or clone the repository
2. Open the folder in VS Code
3. Open `index.html`
4. Right-click and select **Open with Live Server**
